# Silero TTS - Russian Text-to-Speech

Веб-приложение для синтеза русской речи на базе Silero TTS с интерфейсом Gradio.

## Возможности

- **5 голосов**: aidar, baya, kseniya, xenia, eugene
- **Частоты дискретизации**: 8000, 24000, 48000 Гц
- **Управление скоростью**: от 0.5x (медленно) до 4x (быстро)
- **Расстановка ударений**:
  - `model` — автоматически моделью (+ метки)
  - `silero-stress` — через отдельный акцентор (работает с длинными текстами)
  - `manual` — ручная расстановка (используйте `+` для ударения, `ё`)
  - `none` — без ударений
- **Нормализация текста** (через [ru-normalizr](https://github.com/NickZaitsev/ru-normalizr) — Slop-разработка):
  - Аббревиатуры: `ВК` → `вэ ка`, `ГИБДД` → `ги бэ дэ дэ`
  - Латиница: `iPhone` → `айфоун`
  - Числа, даты, время, римские цифры
- **Длинные тексты**: автоматическое разбиение на сегменты
- **GPU/CPU**: автоопределение CUDA

### 2. Установка зависимостей

```bash
pip install -r requirements.txt
```

> **Примечание**: `silero-stress` и `transformers` устанавливаются автоматически через requirements.txt. Для работы silero-stress требуется токенизатор `DeepPavlov/rubert-base-cased` (скачивается автоматически при первом запуске).

## Запуск

```bash
python -m silero_tts
```

Приложение запустится на `http://localhost:7860`

## Использование

1. Введите текст (до 140 символов — одним куском, длиннее — с разбиением)
2. Выберите диктора
3. Выберите частоту дискретизации
4. Настройте скорость воспроизведения
5. Выберите метод расстановки ударений
6. Нажмите **Preview Stress** для предпросмотра текста с ударениями
7. Нажмите **Generate** для генерации аудио

## Как это работает

### Длинные тексты
Тексты длиннее `MAX_CHARS` (по умолчанию 140) разбиваются на части:
- Сначала по абзацам
- Затем по строкам
- Затем по предложениям
- В крайнем случае по словам

Между частями добавляется пауза 0.12 сек.

### silero-stress и длинные тексты
Акцентор использует BERT с лимитом 2048 токенов. Для длинных текстов реализовано токен-ориентированное разбиение (через `DeepPavlov/rubert-base-cased`), чтобы избежать ошибок TorchScript.
